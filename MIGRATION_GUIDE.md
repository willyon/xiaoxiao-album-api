# 人脸聚类功能数据库迁移指南

## 概述

本迁移脚本用于为人脸聚类功能添加必要的数据库字段。

## 需要添加的字段

### 1. face_embeddings 表

- `ignored_for_clustering` (BOOLEAN DEFAULT FALSE) - 标记用户手动移除的照片，避免下次聚类时重新加入

### 2. face_clusters 表

- `name` (TEXT) - 人物名称（用户可编辑）
- `updated_at` (INTEGER) - 更新时间戳

## 使用方法

直接运行迁移脚本：

```bash
node src/migrations/addFaceClusterFields.js
```

## 迁移脚本特性

- ✅ **安全检查**：自动检查字段是否已存在，避免重复添加
- ✅ **幂等性**：可以安全地多次运行，不会出错
- ✅ **错误处理**：如果迁移失败，会显示详细错误信息并退出

## 注意事项

1. **备份数据库**：在生产环境执行迁移前，请先备份数据库
2. **服务停止**：建议在服务停止时执行迁移，避免数据不一致
3. **测试环境**：先在测试环境验证迁移脚本
